# Форум на Фласк

Имплементираме прост уебсайт, използвайки Flask.

## Инсталация

```bash
virtualenv env
source env/bin/activate

pip install -r requirements.txt

export FLASK_APP=main.py
export FLASK_ENV=development
flask run
```
